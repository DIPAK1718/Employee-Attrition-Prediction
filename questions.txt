Q. Why do we check the target variable distribution before training a model?
Interview-Level Answer:
We check the target variable distribution to determine whether the dataset is balanced or imbalanced. This helps us choose appropriate evaluation metrics, sampling strategies, and machine learning algorithms.

Follow-up Question (Very Common)

Q. What is an imbalanced dataset?
Answer:
An imbalanced dataset is one where one class has significantly more samples than the other class.
Example:
No  = 95%
Yes = 5%
The model can achieve high accuracy by predicting only the majority class, making accuracy misleading.

Common Mistake
Many candidates say:

That's not enough.
A stronger answer is:
"The target distribution determines whether I should rely on accuracy or metrics such as precision, recall, F1-score, or ROC-AUC, and whether I need techniques to handle class imbalance."

That shows you're connecting EDA to later modeling decisions.

Queck Question
Q1. Why can't we use Linear Regression directly for binary classification?
Linear Regression predicts continuous values that can be less than 0 or greater than 1. In binary classification, we need probabilities between 0 and 1. Logistic Regression uses the sigmoid function to convert the linear output into a valid probability.

Q2. What is the purpose of the Sigmoid Function?
Interview Answer
The sigmoid function converts any real-valued input into a probability between 0 and 1, making it suitable for binary classification problems.

Q3. Does Logistic Regression predict probabilities or classes first?
Answer
Logistic Regression first predicts the probability of belonging to the positive class. This probability is then compared with a decision threshold (usually 0.5) to produce the final class label.

Example:
Probability = 0.82
↓
Threshold = 0.5
↓
Prediction = Yes

Q4. What is the default decision threshold?
Answer
The default decision threshold is 0.5. If the predicted probability is greater than or equal to 0.5, the model predicts the positive class; otherwise, it predicts the negative class.
Important: The threshold is not fixed. For some business problems (e.g., fraud detection or disease diagnosis), we may choose a different threshold to prioritize precision or recall.

Q5. Why did we increase max_iter to 1000?
Answer
Logistic Regression trains using an iterative optimization algorithm. If the optimizer cannot converge within the default number of iterations, increasing max_iter allows more iterations so the model can find the optimal coefficients and avoid convergence warnings.

Q. What is the difference between predict() and predict_proba()?
Interview Answer
predict() returns the final predicted class labels (such as 0 or 1), whereas predict_proba() returns the probability of each class. These probabilities are useful when adjusting decision thresholds or evaluating model confidence.

Q.Can we compare coefficients directly?
Answer
Only if the numerical features have been standardized (which you did using StandardScaler). Without scaling, coefficient magnitudes are influenced by the units of the features and are not directly comparable.
