# Employee Attrition Prediction using Machine Learning

## 📌 Project Overview

Employee attrition is a major challenge for organizations because replacing experienced employees is expensive and time-consuming.

This project builds a Machine Learning model that predicts whether an employee is likely to leave the company based on HR data.

The objective is to help HR departments identify high-risk employees and take proactive retention actions.

---

## 📂 Dataset

**Dataset:** IBM HR Analytics Employee Attrition Dataset

Target Variable:

- Attrition
  - Yes
  - No

Number of Records:
- 1470

Number of Features:
- 35

## 💼 Business Insights

- Employees working overtime are more likely to leave.
- Lower monthly income is associated with higher attrition.
- Longer tenure generally reduces attrition.
- The model can help HR identify employees at risk of leaving.
